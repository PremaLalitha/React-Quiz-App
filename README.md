# React Quiz App

A simple and interactive quiz application built using React. Users can answer multiple-choice questions and view their score instantly after completing the quiz.

## Features

- Multiple-choice quiz questions
- Instant score calculation
- Interactive and responsive UI
- Smooth user experience

## Technologies Used

- React.js
- JavaScript
- HTML
- CSS
